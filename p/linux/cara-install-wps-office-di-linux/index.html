    Cara Install WPS Office Di Linux

    Linux






    Bagi teman-teman yang baru pindah ke Linux tentu akan merasa gerah ketika tidak bisa mengunakan atau menginstall Microsoft Office karena Microsoft Office hanya dapat di jalankan pada sistem operasi windows.

    Tapi tenang aja kok, jangan panik masih banyak alternatifnya seperti libre Office. Namun pada tutorial kali ini kita tidak akan mengunakan Libre Office di kita akan mengunakan WPS Office.


    Apa itu WPS Office?


    WPS Office merupakan Aplikasi desktop alternatif penganti Microsoft Office. WPS Office dapat di install di berbagai macam sistem operasi baik itu Windows,Mac,dan Linux.

    Dan yang kerenya lagi WPS Office itu gratis untuk di gunakan.

    Baca juga: Instalisasi Microsoft Office Di Lubuntu


    Mengapa mengunakan WPS Office?


    1.Menjadi sebuah alternatif

    – Tentu tidak cuman penguna Linux saja yang bisa menjadikan alternatif namun penguna Windows dan Mac juga dapat menjadikan WPS Office sebagai alternatif penganti Microsoft Office.


    2.Tampilan UI (User Interface)

    – Yang saya suka dari WPS Office adalah tampilan UI nya yang mirip" dengan Microsoft Office namun lebih clean. Dengan tampilan yang menarik dan juga mudah di gunakan tentu bagi teman" yang habis hijarah dari Microsoft Office ke WPS Office tidak akan kesulitan untuk mengunakanya.


    3.100% Gratis

    – Yang terakhir adalah gratis, teman" tidak perlu mengeluarkan uang sedikitpun untuk mengunakan WPS Office karena WPS Office ini gratis namun teman" juga bisa untuk membeli versi preniumnya. Berbeda dengan Microsoft Office yang harus mengeluarkan uang banyak untuk membeli Aplikasi tersebut dan ujung" nya teman" melakukan hal tercela yakni programm bajakan (crak).


    Tidak ada salahnya kan untuk mencoba WPS Office selain gratis dan bisa di gunakan pada sistem operasi kalian.

    Baca juga: Cara Install Hugo Di Windows



    Cara Instalisasi


    Sangat mudah untuk menginstall WPS Office nya, pertama silahkan kunjungi website www.wps.com

    Setelah mengunjungi websitenya lalu pilih sistem operasi yang akan teman" pilih karena di sini kita mengunakan Linux maka pilih Linux.


    Untuk Debian pilih .deb lalu tunggu proses download nya selesai.


    Jika sudah selesai mendownload file WPS Office .deb nya selanjutnya buka terminal lalu ketikan perintah berikut ini.

    Baca juga: Pengenalan HTML


    Oh ya, jangan lupa untuk CD ( Change Directory ) kalian. Di sini saya menyimpan file yang sudah di download tadi ke directory Donwloads.

    $cd Downloads


    Untuk mengisntall masukan perintah berikut dan nama file yang sudah di download tadi biar mudah copas aja.

    $sudo dpkg -i wps-office_11.1.0.9719.XA_amd64.deb


    Tunggu prosesnya selesai maka akan terinstall secara otomatis WPS Office nya.


    Tampilan WPS Office.


    Tersedia beberapa plihan seperti Word,Excel, dan Powerpoint.



    Kekuranganya tidak terinstall font Microsoft seperti Times News Roman dsb.

    untuk menginstall font microsoft cukup ketikan perintah berikut.

    $sudo apt install msttcorefonts

    Tekan Tab => Enter => Yes

    Maka font microsoft akan secara otomotasi terinstall ke dalam WPS Office.



    Cukup sekian dari saya, semoga bermanfaat…
