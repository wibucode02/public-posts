    Cara Install Paru Di Linux

    Linux






    Bagi kamu penguna Linux apalagi masih pemula pasti akan merasa kesulitan dalam menginstall sebuah software" tidak seperti di Windows yang hanya tinggal mengklik kanan saja dan sudah langsung terisntall namun di Linux berbeda.

    Terutama buat kamu yang mengunakan Arch Linux atau turunanya sepert Manjaro,Artix dan sebagainya.

    Untuk Arch Linux berbeda dengan Debian/Ubuntu jika kamu ingin menginstall sebuah software kamu harus melakukan perintah ini terlebih dahulu $sudo pacman -S nama-software

    baca juga sharing pengalaman setelah pindah di linux


    Namun hal ini kurang fleksibel apalagi jika kamu salah mengetikan nama software yang mau di install maka akan terjadi eror. Dan tentu nya kamu tidak dapat memilih list peckage software" yang mau di install.

    Nah maka dari itu saya sangat menyarankan mengunakan “Paru” atau di sebut sebagai “Paru is your standard pacman wrapping AUR helper with lots of features and minimal interaction."

    Paru ini berisi macam” peckage repository dari Aur. Bagi kamu yang belum tau apa itu Aur (Arch User Repository) adalah repository berbasis komunitas untuk pengguna Arch. Jadi di Aur itu berisi macam" repository dari komunitas nantinya kita dapat mengunakan repository nya dalam melakukan instalisasi software di Arch Linux dan turunanya.



    Cara Install Paru


    Untuk instalisasinya sangat mudah kamu hanya perlu Terminal saja lalu ikuti langkah" berikutnya.

    $sudo pacman -S --needed base-devel
$git clone https://aur.archlinux.org/paru.git
$cd paru
$makepkg -si



    Cara Pengunaan


    Carahnya sangatlah simpel, hanya perlu mengetikan

    $paru nama-software


    Hasil:


    – ketikan nomor peckage yang mau di install, contoh saya akan menginstall peckage no 1 maka ketik no 1 di paling bawa repository nya.

    – Nanti akan di minta masukan password, silahkan masukan password kamu lalu enter.

    – Untuk confirmasi penginstallan akan muncul plilihan [Y/n] ketik y jika ingin menginstall dan n untuk membatalkan.

    – Langkah terakhir tunggu proses download dan installisasinya sampai selesai dan cek di window manager kalian.


    Note: Jika terjadi eror ketika proses penginstallan mungkin koneksi internet nya terganggu atau peckage yang mau di install tidak valid jadi silahkan ganti peckage lain yang mau di install sampai bisa.


    Untuk keluar dari Paru ketikan ctr+c di Terminal.
