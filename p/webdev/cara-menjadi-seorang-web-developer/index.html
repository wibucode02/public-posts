    Cara Menjadi Seorang Web Developer Bagi Pemula

    webdev






    Cara Menjadi Seorang Web Developer Bagi Pemula - Bagi kamu yang punya keinginan atau cita-cita menjadi seorang Web Developer yang berpengalaman, tentu hal ini tidaklah mudah untuk di jalani apalagi kamu yang baru mulai atau terjun ke dunia Web Developer ini.

    üê± Nah pada kesempatan kali ini saya akan membangikan beberapa tips bagi kamu yang ingin menjadi seorang Web Developer namun saya tidak akan membahas secara detail karena dunia Web Developer itu sangat luas sekali dan banyak hal-hal yang perlu di pelajari, jadi akan saya ringkas saja agar juga bisa di pahami untuk pemula yang benar-benar baru terjun ke dalam dunia Web Developer ini.



    Mengapa Web Developer ?


    Di jaman sekarang yang teknologinya semakin hari semakin berkembang. Teknologi web juga sering di kembangankan guna membantu pekerjaan kita atau kebutuhan sehari-hari. Contoh nya seperti marketplace tokopedia,shopee,lazada kita dapat mengakses berbelanja dan menjual produk kita melalui toko online dengan sangat mudah tanpa harus membuka toko fisik.

    Nah di sini sudah terlihat kan kegunaan menbangun sebuah website seperti marketplace yang saya sebutkan tadi hanya dari website saja kita sudah dapat membeli suatu produk ataupun menjualnya dengan mudah. Tidak hanya marketplace saja namun website yang sering kita akses seperti berita,streaming vidio,social media dan sebaginya semua dapat kita akses dengan mudah melalui website.

    Jadi peluang untuk menjadi seorang Web Developer di jaman sekarang sangat besar dan tentunya banyak di minati oleh orang-orang bahkan perusahaan juga. Dengan adanya peluang besar seperti ini kamu tidak perlu ragu lagi untuk menjadi seorang Web Developer, kamu dapat membangun sebuah website milik kamu sendiri atau membuat website untuk orang lain.

    Dan untuk perkerjaanya kamu dapat mencari job di beberapa situs freelancer atau juga kamu dapat melamar di sebuah perusahaan seperti startup dan sebagainya. Karena mereka juga pasti membutuhkan seorang Web Developer entah itu untuk membangun bisnis atau project.


    Hal Apa Saja Yang Perlu Di Perlukan


    Menjadi seorang Web Developer apa saja yang harus kit persiapan.

    • 1 Koneksi internet - tentu saja menjadi hal yang wajib di miliki
    • 2 Device yang mempenuni - kamu dapat mengunakan laptop atau pc namun jika tidak punya sebenarnya bisa mengunakan smartphone tapi tentunya memiliki kekurangan
    • 3 Mentor/pembimbing - ini juga di perlukan karena kalau belajar sendiri secara otodidak kamu bakal kesulitan misalnya terjadi eror kamu tidak dapat menayankan solusi perihal tersebut tapi kamu juga dapat menyangkan di forum-forum programmer tekenal seperti stackoverflow
    • 4 Bootcamp - ini sebenarnya tidak wajib karena semua orang tidak harus mengikuti bootcamp apalagi mungkin terhalang kesibukan atau tidak mempunyai dana namun dengan mengikuti bootcamp kamu dapat belajar lebih terstruktur dan juga di bimbing setelah lulus dari bootcamp ada kemungkinan menemukan peluang kerja
    • 5 Doa dan niat - jika kamu benar-benar ingin menjadi seorang Web Developer kamu harus mempunya tekat yang kuat dan tidak mudah menyerah

    Jangan jadi suatu alasan apapun itu untuk menghambat kamu untuk mengapai apa yang kamu inginkan selama kamu terus berusaha pasti akan menemukan hasilnya.


    Jalur belajar


    Kamu dapat mempelajari materinya di internet karena sudah banyak tutorial yang beredar di internet dan dapat kamu akses secara mudah atau kamu juga dapat ikut pelatihan kusus seperti bootcamp.

    Dalam dunia Web Developer terdapat 2 bagian yakni front-end dan back-end dan jika dapat menguasai ke 2 hal tersebut di sebut sebagai fullstack namun tidak semua orang dapat menjadi seorang fullstack Web Developer hal ini tidaklah mudah harus benar-benar paham apa yang di gunakan dan perlu di kuasai.

    Front-end bertugas pada tampilan website seperti tampilan UI ( User Interface ) mempercantik tampilan sebuah wesbite mulai dari tata letak layout,warna,typografi,animasi dan sabagainya. Tidak hanya di UI nya saja namun juga berperan penting dalam membuat website jadi lebih interaktif.

    Hal yang wajib di kuasai
    Html - CSS - Javascript

    Framework
    CSS - Bootstrap - Tailwind - Materialize
    Javascript - React - Angular - Vue


    Back-end berbeda dengan front-end yang mengurus tampilan web. Back-end tugas nya untuk membuat website menjadi lebih dinamis, back-end tidak fokus pada tampilan website namun lebih pada data sebuah sistem yang nantinya akan di proses contoh nya ketika kita membuat akun baru di sebuah website memasukan nama,email dan juga password akan otomatis tersimpan di website tersebut agar kita dapat login kembali tanpa harus membuat akun lagi.

    Hal yang wajib di kuasai
    PHP - Mysql

    Framework
    CodeIgniter - Laravel - Yii
