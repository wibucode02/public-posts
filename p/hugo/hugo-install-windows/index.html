    Cara Install Hugo Di Windows

    Hugo






    Hi, pada kesempatan kali ini saya akan membuatkan tutorial Cara isntall Hugo di Windows. Sebelumnya saya sudah membahas pengenalan mengenai Hugo bagi yang belum baca klik di sini

    Sebernanya untuk install Hugo di sistem operasi Windows sedikit ribet perlu konfigurasi manual, berbeda dengan sistem operasi lainya yang sudah secara auto jadi tidak perlu di setup manual.

    Namun tenang aja kok ga terlalu sulit hanya perlu step by step saja dan sekali install sudah bisa di jalankan pada sistem operasi Windows. Oh yah tutorial ini tidak hanya untuk Windows 10 saja namun bisa juga di Windows 7 dan caranya sama aja.



    Cara setup installer


    Apa saja yang di perlukan?
    – Terminal
    – Installer Hugo

    Pertama silahkan buka website ini (https://github.com/gohugoio/hugo/releases) lalu pilih setup installer hugo windows 64bit, bagi kalian yang mengunakan 32bit silahkan menyesuaikan, namun di sinii saya mengunakan 64bit kemudian di download.


    Setelah di download kemudian di extra, lalu silahkan teman" pindahkan file installer bebas mau di data mana, kalau saya di E:\hugo\

    Langkah selanjutkan ketikan Edit the system environment variabels, di search cortana windows.

    Pilih Advanced => Environment Variabels => Path => Edit .

    Sesuaikan dengan Path punya kalian ya lalu tinggal ok ok aja. Jika belum punya buat baru di pojok kanan atas menu New.

    Langkah terakhir buka Terminal/CMD lalu ketikan hugo version. Tapi sebelum itu Cd dulu ke directory hugo nya E:\hugo kemudian enter.

    hugo version

    Jika berhasil maka akan muncul seperti ini di Command Prompt nya:

    Dan selamat Hugo berasil terinstall di sistem operasi kalian.

    Note: Jika masih belum berhasil coba cek di bagian Path Edit environment tadi dan pastikan Path nya sudah benar. Kalau masih belum bisa coba close CMD nya lalu buka lagi dan jalankan perintah hugo version.
