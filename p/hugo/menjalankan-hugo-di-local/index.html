    #1 Menjalankan Hugo Di Local

    Hugo






    Masih berlanjut pembelajaran Hugo kita, sebelum nya sudah membahas cara install hugo pada sistem operasi windows klik di sini bagi yang belum baca.

    Lasung saja ke tutorialnya bagi teman" yang sudah menginstall Hugo di sitem operasi baik itu windows,linux,mac.

    Yang pertama adalah membuat nama project Hugo kita sebagai contoh akan saya beri nama belajar-hugo.


    $hugo new site belajar-hugo


    Nah silahkan cek di filemanager teman" akan otomatis di buatkan folder belajar-hugo yang di buat tadi secara otomatis.

    Perlu di ingat memang sudah bisa menjalankan local server namun sebelum itu kita perlu menginstall themes nya dulu agar kita bisa melihat perubahan di Hugo ketika di edit.

    Tenang aja di sini kita mengunakan themes yang sudah di sediakan oleh Hugo jadi tidak perlu membuat themesnya secara manual, kalau teman" ingin membuat themes hugo dari awal juga tidak masalah.

    Jangan lupa untuk CD " Change Directory "


    $cd belajar-hugo
$git init
$git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke


    Perlu di ingat bilah terjadi eror pastikan teman" sudah menginstall Git.

    Tunggu proses cloning template Hugo nya, lalu cek di folder kalian akan di buatkan folder themes yang kita tambahkan tadi.

    Lalu pergi ke file config.toml dan tambahkan theme = “ananke”


    baseURL = "http://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"


    Dan jangan lupa untuk di save.

    Langkah terakhir adalah membuat posts atau isi kontent yang akan di generate oleh Hugo, file yang akan di buat adalah file markdownya.


    $hugo new posts/belajar-hugo.md


    Rubah true menjadi false di file markdown yang sudah di buat tadi dan silahkan ketikan apa saja yang mau di jalankan pada local server nya.

    Sekali lagi saya ingatkan di Hugo sudah serba otomatis ketika ingin membuat folder atau file nya.

    Tahap terakhir adalah menjalankan server local Hugo nya agar nantinya kita dapat melihat perubahan yang sudah di buat tadi.


    $hugo server


    Cek di terminal teman" akan secara otomatis di buatkan local server nya.


    http://localhost:1313/


    Dan paste ke web browser maka hasilanya akan seperti ini :


    Itu saja cara menjalankan hugo di local. Cukup mudah kan bagi teman" yang masih awam dengan terminal emang agak susah namun akan terbiasa kok…
