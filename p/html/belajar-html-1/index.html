    Belajar Html 1

    Html






    Pada materi sebelumnya kita sudah belajar mengenal siapa sih penemu Html dan tujuan pembuatanya untuk apa, bagi teman-teman yang belum membacanya klik here

    Nah sekarang kita akan belajar mengenai struktur pada Html ada apa aja sih?

    Buat teman-teman sekalian yang benar" baru belajar tidak masalah kok, di sini kita sama-sama belajar mulai dari awal hingga teman" semuanya dapat mengerti dengan mudah.


    Ini merupakan struktur dari Html, bagi teman" yang baru pertamakali melihat ini tentunya akan bingung.

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>


    Saya akan menjelaskan satu persatu mengenai syntax" pada Html yang sudah di buat tadi.

    <!DOCTYPE html>

    Menunjukan bahwa ini adalah document Html.


    <html lang="en">

    Bahasa pada Html di sini mengunakan bahasa inggris tentunya.


    <head>

    Bagian kepala atau tempat di mana kita menyimpan Meta tag, link css, title dsb.


     <meta charset="UTF-8">

    Universal Character Set, atau yang umum disebut sebagai Charset adalah kumpulan dari beberapa jenis pengkodean karakter baik huruf, angka, symbol, dll.


    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    Menyesuiakan dengan device yang akan di buka pada brower mobile.


     <title>Document</title>

    Merupakan judul dari Html tersebut ketika di buka pada browser, makan akan menunjukan judul yang akan di buat.


    <body>

    Badan dari Html di sini kita dapa memasukan syntax html seperti class,id dsb.


    </head>

</body>

</html>


    Untuk tag penutup merupakan akhir dari tag pada html, jika kita menuliskan contoh sebuah heading 

     <h1>Heading</h1>
    maka harus di awali dengan tag pembuka <> dan kemudian tag penutup </> ini merupakan validasi dari Html agar syntax yang di buat dapat di jalankan pada browser.



    Selanjutnya kita akan membuat syntax Html dan menjalankan nya pada browser…
