    Bahasa Pemrograman Yang Populler Di Indonesia

    Rekomendasi






    Perlu kita ketahui banyak bahasa pemrograman yang di buat guna membantu pekerjaan manusia, namun sebagian orang jarang mengunakan bahasa pemrograman tersebut ada juga yang lebih memilih mengunakan bahasa pemrograman yang di sukai.

    Banyak programmer" yang baru terjun ke dunia coding namun kebanyakan dari mereka sulit untuk memilih bahasa pemrograman yang di minati. Ada juga yang hanya mengikuti tren saja, namun tidak benar" bisa mengusainya.

    Maka dari itu saya akan memberikan beberapa rekomendasi bahasa pemrograman yang banyak di minati kuhususnya programmer indonesia.


    Javascript

    Javascript merupakan bahasa pemrograman yang paling populler. Tidak hanya populler di indonesia saja namun sangat populler di seluruh dunia.

    Belum lagi framework" javascript yang populler seperti React,Vue,Angular dan masih banyak lagi.

    Javascript bahasa pemograman yang sangat interaktif kita juga dapat membangun sebuah website seperti membuat animasi dll, hanya mengunakan javascript dan javascript tidak hanya untuk website saja namun bisa juga untuk membangun sebuah Aplikasi mobile tentunya dengan mengunakan framework seperti React Js.

    Oh ya, jika kamu ingin menjadi seorang front-end web kamu juga wajib untuk mempelajari Javascript.


    PHP

    Dan yang populler ke 2 tentunya adalah PHP apalagi untuk programmer yang main di belakang ( back-end ) tentunya PHP menjadi salah satu bahasa pemrograman yang cukup di minati khususnya programmer indonesia.

    Dengan mengunakan PHP kita dapat membangun sebuah website dinamasi yang nantinya akan lebih berfokus pada struktur data pada website yang akan di kelolah.

    PHP juga merupakan bahasa pemograman yang mudah di pelajari apalagi untuk pemula dan untuk menjalankan program PHP di perlukan web server agar nantinya perintah yang akan kita buat mengunakan PHP dapat di eksekusi melalui web sever.


    Python

    Selanjutnya adalah Python ( ular:V ) yup bahasa pemrograman yang satu ini tidak kalah menarik dari bahasa" pemrograman yang lainya. Tentunya sangat di minati oleh programmer indonesia. Mengapa demikian?

    Karena Python merupakan bahasa pemrograman yang paling mudah di gunakan serta sangatlah simpel tidak seperti bahasa" pemrograman yang lainya yang banyak aturan mainya.

    Tentu bagi pemula Python sangatlah di rekomendasikan apalagi bagi kalian yang masih bingung mau mulai belajar bahasa pemrograman mulai dari mana, Python menjadi salah satu pilihan.


    Java

    Karena tadi sudah bahasa yang mudah sekarang bagian yang susahnya hehe..

    Java ya bukan ( Jawa ) itu beda lagi. Java merupakan bahasa pemrograman yang cukup sulit untuk di pelajari apalagi untuk pemula. Namun walaupun sulit Java juga cukup populler di kalangan programmer indonesia.

    Apasih yang dapat di buat mengunakan Java?

    Java kebanyakan di gunakan untuk membuat sebuah aplikasi mobile berbasis Android. Bagi para Android Developer banyak mengunakan bahasa pemrograman Java untuk membuat aplikasi mobile.

    Walapun sulit bukan berarti kita tidak dapat mempelajari bahasa pemrograman yang satu ini, asal ada kemauan dan niat pas bisa kok.


    C

    C,C#,C++, merupakan bahasa pemrograman yang populer juga tidak hanya di indonesia namun di luar sana banyak yang mengunakan bahasa pemrograman ini.

    Kita dapat membuat aplikasi dekstop ataupun membuat sebuah game yang tidak hanya untuk game" PC namun bisa juga untuk Mobile.

    Ketika saya membahasa bahasa pemrograman ini saya jadi kepikiran game, ya sebab bahasa pemrograman ini mungkin lebih fokus ke sana. Sebenarnya tidak hanya untuk membuat game saja namun masih banyak yang dapat di buat dengan mengunakan bahasa pemrograman ini.


    Dart

    Pasti kalian agak asing dengan bahasa pemrograman ini, dan mengapa saya memasukan ke list sebagai bahasa pemrograman yang populler di indonesia.

    Dart merupakan bahasa pemrograman yang di buat oleh google, terus kegunaanya apa?

    Maka saya akan menjawap “FLutter” Flutter merupakan framework yang di buat oleh google untuk membuat aplikasi mobile berbasis Android dan juga IOS hanya dengan mengunakan Flutter dapat melakukan ke 2 hal tersebut.

    Nah Flutter ini mengunakan bahasa pemrograman Dart yang nantinya di gunakan untuk membuat sebuah aplikasi mobile Android dan IOS.



    Nah itu saja yang dapat saya bahas, ini berdasarkan pendapat saya jadi mungkin ada keliruan atau ada beberapa bahasa pemrograman lain yang tidak saya masukan ke list bisa teman" coment di kolom komentar…
