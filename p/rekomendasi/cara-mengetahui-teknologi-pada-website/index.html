    Cara Mengetahui Teknologi Pada Website

    Rekomendasi






    Pernah ga sih ketika kalian mengakses sebuah website dan kalian penasaran teknologi apa saja yang di gunakan untuk membangun website tersebut. Terkadang memang kita bisa menebak-nebak teknologi di balik website yang kita akses ada banyak cara salah satuhnya dengan inspect element namun tentu hal tersebut kuranglah akurat apalagi untuk website" dinamis.

    Nah, di kesempatan kali ini saya akan memberikan cara agar kita dapat mengetahui teknologi-teknologi apa saja yang di gunakana pada website tersebut.



    Di sini terdapat 2 cara


    Melalui website builtwith

    builtwith merupakan website yang cukup populler di gunakan dalam hal mengetahui teknologi apa saja yang di gunakan pada website dan juga seperti :

    1.Web server
    2.Nameserver Providers
    3.SSL Certificate
    4.Hosting Providers
    5.Email Services
    6.Frameworks, Analytics and Tracking
    7.Content Management Systems
    8.Server Information


    Caranya cukup mudah kok hanya perlu mengunjungi web nya di builtwith.com

    Kemudia masukan nama website lalu enter.




    Mengunakan Chrome Extensions

    Cara ini paling praktis dan juga simpel tanpa perlu mengunjungi website seperti builtwith, karena ini merupakan Extensions pada Google Chrome jadi hanya dapat di gunakan pada browser tersebut dan tidak dapat di gunakan pada browser lain.

    Nama Extensions nya Wappalyzer silahkan teman" install. Bagi yang belum tau cara install nya Cara Mengunakan Extensions Pada Google Chrome

    Jika sudah di install silahkan buka website yang ingin teman" lihat teknologi nya contoh saya mengunakan website wibucode.com, teman" juga bebas mau mengunakan website apa.

    Untuk cara pengunaan klik ikon di pojok kanan atas seperti pada gambar berikut.



    Sangat akurat dapat mengetahui framwork apa saja yang saya gunakan dan juga hosting nya di mana. Keren sekali dan sangat rekomendasi untuk di gunakan.
