    Cara Mengunakan Extensions Pada Google Chrome

    Rekomendasi






    Apa itu Extension pada Google Chrome ?

    Extensions merupakan tools yang dapat kita gunakan agar membantu pekerjaan/kebutuhan kita dan Extension yang saya bahas kali ini merupakan Extension pada Google Chrome jadi Extensions tersebut hanya dapat di gunakan pada browser Google Chrome saja dan tidak dapa di gunakan pada web browser lainya.

    Caranya sangat mudah kalian hanya perlu mengunakan Google Chrome untuk mengunakan Extensions.

    Silahkan ikuti langkah-langkah berikut ini :


    Buka menu setting pada Google Chrome.


    Kemudian ke menu Extension.


    Maka akan muncul Extensions" yang sudah terinstall pada Google Chrome kalian lalu klik menu navbar di pojok kiri => Open Chrome Web Store.


    Silahkan ketikan Extension yang mau di isntall contoh saya akan menginstall Extension Zoom.


    Jika sudah ketemu, klik Add to Chrome.


    Kemudian Add extensions


    Dan Extensions akan secara otomatis terinstall, untuk cara mengunakankan cukup klik ikon di pojok kanan atas.



    Itu saja cara menginstall extension pada google chrome,cukup mudah bukan…
