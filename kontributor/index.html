    Cara Berkontributor Di Wibucode







    Bagi teman-teman yang ingin berkontributor sangatlah mudah. Karena Wibucode merupakan komunitas Open source, jadi siapa saja dapat berkontributor bersama kami dan gratis…


    Disini terbagi beberapa kontributor yakni:

    • Desing UI
    • Developer web/mobile
    • Penulis


    Hal yang wajib di butuhkan dari kalian ada 2 yaitu Nama dan Fotoprofil agar nantinya postingan yang di buat mengunakan Nama dan Foto kalian. Jika tidak punya/tidak mengisi maka kami akan mengunakan Nama default dan Foto default..

    Usahakan sebisanya ada.

    Note penting: Untuk ukuran thubnail width 640 x height 360



    Design UI


    Kalian bisa mengirimkan Desain UI yang sudah kalian buat baik itu Web dan juga Mobile App.

    Syarat :

    – Memiliki file img dengan format jpg atau png
    – Mengunakan karya yang di buat original ( punya sendiri )
    – Mencantumkan deskripsi misalnya tujuan di buatnya atau refrensi pembuatanya
    – Mencantumkan Tools dan Aset yang di gunakan
    – Bisa juga mencatumkan link media social punya kalian atau link refrensi desain yang di buat


    Developer web/mobile


    Kalian bisa mengirimkan Project ngoding yang sudah kalian buat baik itu Web dan juga Mobile App.

    Syarat :

    – Memiliki file img dengan format jpg atau png
    – Mengunakan karya yang di buat original ( punya sendiri )
    – Mencantumkan deskripsi misalnya tujuan di buatnya atau refrensi pembuatanya
    – Mencantumkan Tools dan Aset yang di gunakan
    – Mencantumkan Demo/Preview dari project yang di buat jika punya, jika tidak maka di gantikan dengan link repository
    – Bisa juga mencatumkan link media social punya kalian atau link refrensi desain yang di buat


    Penulis


    Ingin jadi penulis artikel, namun tidak memiliki hosting dan domain. Tenang aja hal tersebut dapat di kabulkan. Kalian bisa mengirimkan hasil tulisan kalian ke website kami.

    Syarat :

    – Mempunyai gambar thubnail jika puya, jika tidak maka akan di buatkan
    – Memiliki file img dengan format jpg atau png
    – Mengunakan tulisan yang di buat original ( punya sendiri )
    – Mencantumkan deskripsi artikel jika punya
    – Bisa juga mencatumkan link media social punya kalian atau link refrensi artikel yang di buat


    Cara submit


    Ada banyak cara untuk mengirimkanya tentu sangatlah mudah.

    Yang pertama bisa mengirimkannya secara langsung ke akun social media kami seperti facebook,instagram,telegram,dan discord atau di kolom komtar disqus.

    Untuk format file yang di gunakan bisa txt,pdf,word,dan markdown.

    Melakukan pull request ini juga salah satu rekomendasi jika ingin di revisi kembali. Untuk cara ini harus berformat .md ( Markdown ) jika tidak maka tidak akan di terima.

    – Caranya cukup mudah silahkan fork repository ini https://github.com/wibucode02/tempat-kontributor


    – lalu clone dan pull


    untuk format Markdwonya seperti ini :

    ---

author: " Nama kamu "

title: " Judul arikel "

description: " Deskripsinya "

---



    Itu saja bagaimana cara untuk berkontributornya jika ada yang ingin di tanyakan silahkan…
